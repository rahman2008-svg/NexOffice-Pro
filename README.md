📘 NexOffice Pro
A modern offline Android productivity suite built with Kotlin and Jetpack Compose.
🚀 About the Project
NexOffice Pro is a lightweight, offline-first office productivity application designed for Android users.
It provides tools for documents, sheets, slides, PDFs, and file management in one unified app experience.
